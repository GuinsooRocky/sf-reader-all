x-reader — 三层架构
按需组合，不用的层不影响使用
Layer 1 · 基础层
Python CLI / Library
pip install 即用。10+ 平台自动识别 → 抓取 → 统一 Markdown 输出 → 存入 Obsidian
$ x-reader https://mp.weixin.qq.com/s/abc123
↓ 可选扩展
Layer 2 · AI 增强层
Claude Code Skills
复制 skills/ 到 ~/.claude/skills/ 即可。发视频/播客链接 → 自动 Whisper 转写全文 → 结构化摘要 + 行动清单
↓ 可选扩展
Layer 3 · 工具层
MCP Server
4 个工具接口：read_url · read_batch · list_inbox · detect_platform — 接入任意 AI 工作流
